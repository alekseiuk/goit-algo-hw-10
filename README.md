# Домашнє завдання: Обчислення інтеграла методом Монте-Карло

## Опис завдання
Метою завдання було обчислення значення визначеного інтеграла функції $f(x) = x^2$ на інтервалі $[0, 2]$. 
Для цього було використано два підходи:
1. **Метод Монте-Карло** (імовірнісний підхід).
2. **Аналітичний розрахунок** (використання функції `quad` з бібліотеки `scipy.integrate`).

## Результати обчислень

### 1. Метод Монте-Карло
Для розрахунку було згенеровано **1 000 000** випадкових точок у прямокутнику, що обмежує функцію.
- **Отриманий результат:** `2.665704`

### 2. Аналітичний розрахунок (функція `quad`)
Функція `quad` використовує чисельні методи інтегрування з високою точністю, що є еталонним значенням для порівняння.
Математично точне значення:
$$\int_{0}^{2} x^2 \,dx = \left[ \frac{x^3}{3} \right]_{0}^{2} = \frac{8}{3} \approx 2.666666...$$
- **Отриманий результат (`scipy.integrate.quad`):** `2.666666666666667`

## Порівняння та аналіз похибки

Щоб оцінити точність методу Монте-Карло, обчислимо абсолютну та відносну похибки:

1. **Абсолютна похибка** ($E$):
   $$E = |V_{exact} - V_{MC}|$$
   $$E = |2.666667 - 2.665704| = 0.000963$$

2. **Відносна похибка** ($\varepsilon$):
   $$\varepsilon = \frac{E}{V_{exact}} \times 100\%$$
   $$\varepsilon = \frac{0.000963}{2.666667} \times 100\% \approx 0.036\%$$

## Висновки

1. **Правильність розрахунків:** Метод Монте-Карло показав результат, дуже близький до аналітичного. Різниця починається лише у третьому знаку після коми.
2. **Точність методу:** При використанні 1 000 000 випадкових точок відносна похибка склала всього **0.036%**. Це свідчить про те, що алгоритм реалізовано правильно, і він забезпечує достатню точність для більшості прикладних задач.
3. **Особливість методу:** Хоча метод Монте-Карло є простим у реалізації та дозволяє інтегрувати функції, які важко обчислити аналітично, він потребує великої кількості вибірок (ітерацій циклу) для досягнення високої точності.
